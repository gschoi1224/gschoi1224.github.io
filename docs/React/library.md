# 유용한 라이브러리
{: .fw-700 .fs-7 }

## react-outside-click-handler

> OutsideClickHandler 컴포넌트를 사용해 외부 클릭 시 원하는 이벤트를 넣을 수 있는 기능을 제공

```js
import OutsideClickHandler from 'react-outside-click-handler';

<OutsideClickHandler onOutsideclick={외부 클릭 시 이벤트}
<button />
</OutsideClickHandler>
```
